# MineUI: Minecraft Server Panel

# Install
**Using install.sh**
```
$ bash install.sh
```

# Install Manually
**Setting the server path.**
```
$ touch .settings

$ echo /path/to/server/ > .settings
```

**Create a softlink.**
```
$ ln -s /path/to/server/logs
```

**Give `.mcsrv.sh` execution permission.**
```
$ sudo chmod +x .mcsrv.sh
```

**Done! ^-^**
